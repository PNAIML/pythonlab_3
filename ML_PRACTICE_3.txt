Identifying Scenarios for supervised learning

A company wants to predict the future sales of their product based on past sales data. Which machine learning model should they use?
 Classification
 Regression
 Clustering
 Dimensionality Reduction

Accepted Answers:
Regression

A hospital wants to determine if a patient has a disease based on various health metrics. Which machine learning model is appropriate?
 Regression
 Classification
 Clustering
 Dimensionality Reduction

Accepted Answers:
Classification

In a supervised learning task, the goal is to predict a continuous output variable. Which of the following would be appropriate?
 Decision Tree Classifier
 K-Means Clustering
 Linear Regression
 Principal Component Analysis

Accepted Answers:
Linear Regression

A researcher is using historical temperature data to predict future temperatures. What type of problem is this?
 Classification
 Regression
 Clustering
 Reinforcement Learning

Accepted Answers:
Regression

A bank wants to predict whether a loan applicant will default on a loan. Which type of model should they use?
 Regression
 Classification
 Clustering
 Association Rule Learning

Accepted Answers:
Classification

A medical diagnosis system needs to determine if an X-ray image shows signs of pneumonia. What type of machine learning model should be used?
 Regression
 Classification
 Clustering
 Association Rule Learning

Accepted Answers:
