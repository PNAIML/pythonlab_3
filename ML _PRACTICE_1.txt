
Q.Which of the following best describes machine learning?
 A technique for programming computers to perform specific tasks
 A method for analyzing data to make predictions or decisions without explicit programming
 A system for storing and retrieving large amounts of data
A way to design user interfaces for applications











Accepted Answers:
A method for analyzing data to make predictions or decisions without explicit programming


Q.Using historical sales data to forecast future sales is an example of:
 Data Storage
 Machine Learning
 Network Security
 Software Development









Accepted Answers:
Machine Learning

Q.Training a model to recognize speech patterns and convert spoken words into text is an example of:
 Data Encryption
 Machine Learning
 Database Management

Accepted Answers:
Machine Learning
